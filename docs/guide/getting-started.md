# Guide de démarrage

## Prérequis

- Node.js 20+
- Python 3.11+
- uv

## Installation

```bash
git clone <repo>
cd blueprint-ai-stack
npm install
```

## Lancer la documentation

```bash
npm run docs:dev
```

Ouvrir `http://localhost:5173`.

## Workflow

1. Rédiger une spec dans `specs/`
2. Créer un ticket Plane
3. Développer avec OpenCode
4. Valider et merger
