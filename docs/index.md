# Blueprint AI Stack

**Framework de développement augmenté par l'IA.**

Blueprint AI Stack orchestre un ensemble d'outils spécialisés pour accélérer la conception, la validation et le déploiement de projets logiciels.

- **Spec Kit** — Spécifications et cadrage
- **OpenCode** — Développement local assisté
- **Plane** — Gestion de projet et tickets
- **n8n** — Orchestration et automatisation
- **Graphify** — Carte de connaissance
- **VitePress** — Documentation des specs
- **GitHub** — Code source et CI/CD

> Ce dossier est un **template maître**. Clonez-le pour chaque nouveau projet.
