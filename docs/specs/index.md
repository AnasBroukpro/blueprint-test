# Spécifications

Cette section contient les spécifications techniques et fonctionnelles du projet, rédigées via Spec Kit.

## Créer une spécification

1. Définir le besoin
2. Rédiger la spec dans `specs/<feature>/spec.md`
3. Valider avec la checklist qualité
4. Créer le ticket Plane correspondant

## Structure

```
specs/
├── <feature>/
│   ├── spec.md        # Spécification
│   ├── plan.md        # Plan d'implémentation
│   ├── tasks.md       # Découpage en tâches
│   └── checklists/    # Check-lists de validation
```
